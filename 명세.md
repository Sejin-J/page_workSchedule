월간 근무표 자동 생성 웹 프로그램 요구사항 명세서 (v2.0)
1. 프로그램 개요
목적: 구성원들의 숙련도, 부재 일정, 근무 선호도를 종합하여 '근무의 균등한 분배'를 최우선으로 하는 월간 스케줄 자동 생성.

실행 환경: GitHub Pages (서버가 필요 없는 HTML/CSS/JavaScript 및 Web Worker 기반의 단독 구동 정적 웹 앱).

2. 입력 명세 (Input)
2.1. 환경 및 파라미터 설정 (Web UI)
대상 연/월: 스케줄을 생성할 기준 연도와 월 지정.

휴일 설정: 주말 및 공공 API 기반 공휴일 자동 지정, 달력 UI를 통한 사용자 지정 휴일 수동 토글.

[수정] 기본 필요 인원: 근무 종류별(A, B, C) 및 요일별 일일 최소 필요 인원.

특정일 예외 인원 (Daily Override): 특정 일자에 한해 기본 필요 인원을 무시하고 덮어쓰는 기능.

최대 연속 근무 제한: 개인별 쉬지 않고 연속 출근할 수 있는 최대 일수 제한.

2.2. 제원 및 이력 데이터 (CSV 업로드)
[추가] 직책 및 기본 정보: 식별자(이름/사번), 직책(팀장/일반), 성숙도 (1:초급, 2:중급, 3:고급).

[수정] 근무 선호도: A, B, C 근무에 대한 1~3순위 지정 (팀장은 입력 불필요).

근무 가능 기간: 중간 입/퇴사자 대응을 위한 스케줄 배정 시작/종료일.

부재 일정: 당월 연차, 교육 등 배정 불가 날짜 목록.

사전 지정 근무 (Fixed): 특정 일자 및 근무에 고정할 인원 명단.

[수정] 전월 이월 데이터: 전월 마지막 주 일일 스케줄 및 전월까지의 누적 할당 횟수(종류별, 월수토별).

2.3. 업로드 데이터 사전 검증 (Pre-flight Validation)
최적화 실행 전 CSV 데이터의 형식/논리적 오류(선호도 중복, 필수 값 누락 등) 검사 및 UI 에러 피드백.

3. 출력 명세 (Output)
웹 화면 표출: 부서원 전체 매트릭스 형태의 종합 근무표, 개인별 달력 및 통계/검증 뷰.

파일 다운로드: 마스터 표(시트 1) 및 개인별 캘린더/통계(시트 2) 구조의 파일 저장 (Excel/CSV).

4. 필수 제약 조건 (Hard Constraints)
단 하나라도 위배될 경우 해당 스케줄은 '생성 불가(Infeasible)' 처리되는 절대 규칙입니다.

최소 인원 충족: 일자/근무별 최소 필요 인원 충족 (특정일 예외 인원 최우선 적용).

사전 지정 근무 고정: 확정 배정(Fixed) 인원은 알고리즘 변경/취소 불가.

[추가] 팀장 전담 룰: * '팀장' 직책을 가진 인원은 오직 'A' 근무에만 배정됨 (B, C 근무 배정 절대 불가).

'팀장'은 한 달 동안 월, 수, 토요일에 해당하는 일자에 최소 7회~8회 이상 필수적으로 출근(A근무)해야 함.

성숙도 안전 배정: 동일 근무조에 성숙도 1(초급) 단독 배정 불가 (반드시 2 또는 3 동승).

배정 불가 조건 엄수: 1일 2개 이상 근무 중복 불가, 부재일(휴가) 및 근무 가능 기간 외 날짜 배정 불가.

[수정] C 근무 전환 금지: C 근무를 수행한 다음 날에는 A, B 근무 배정 절대 불가 (C 근무 연속 수행 또는 휴식만 허용).

최대 연속 근무 제한: 파라미터의 최대 연속 출근 일수 초과 불가 (전월 데이터 연계).

5. 최적화 목적 함수 (Soft Constraints / Priority)
필수 제약을 만족하는 조건 내에서, 아래의 계층적 우선순위에 따라 최적 해를 탐색합니다.

[수정] [우선순위 1] 비례 할당 기반의 절대 균등화:

구성원의 '근무 가능 일수'에 비례하여 목표 할당량 개별 산출.

구성원 간 '전체 누적 근무 횟수 편차(Variance)' 최소화.

고강도 요일(월, 수, 토) 출근 균등화: 팀장을 제외한 나머지 일반 인원들 사이에서, 업무 강도가 높은 월/수/토요일에 배정되는 총 횟수가 특정인에게 몰리지 않고 고르게 분배되도록 최우선 최적화.

[우선순위 2] 균등성 내 선호도 극대화:

균등 분배를 훼손하지 않는 범위에서 1·2순위 지망 근무 배정 최대화.

[수정] [우선순위 3] 휴일 분산 및 피로도 최소화:

개인의 휴일(비근무일)이 특정 주차에 몰리지 않고 고르게 분산.

C 근무 다음 날 '연속 C 근무' 보다는 '휴식' 배정에 가점 부여.

6. 예외 처리 및 UI 편의 기능 (Fallback & UX)
작업 상태 캐싱: 브라우저 로컬 스토리지 자동 저장.

탐색 실패 원인 안내: 충돌 원인 팝업 안내 및 미완성 초안(Fallback) 강제 출력 지원.

수동 편집 및 검증: 웹 화면 내 드래그 앤 드롭(Swap) 지원, 제약 위배 시 실시간 붉은색 경고.

편집 제어: [◀ 뒤로 돌리기], [▶ 앞으로 돌리기] 버튼 UI 제공.

인쇄 최적화 뷰: UI 요소를 숨긴 종합 근무표 A4 규격 출력 지원.
